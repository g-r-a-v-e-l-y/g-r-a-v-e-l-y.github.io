acidstep:

My first Litestep theme, minimal and functional with all original graphics and UI elements.


The extreme bottom left contains a row of shortcut icons that open programs when selected. The double-box above that displays the contents of the multiple desktops used in my theme - virtual desktops. to the right of the virtual desktop display is a lone icon representing the system tray. The menu shown in the bottom right is a semi-transparent program launching popup menu brought up by right clicking anywhere on the desktop. The bars along the top of the screen shot show the current running programs or tasks. All images, including the background, or "wallpaper" are original.
